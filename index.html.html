📌 Description

GVG League est un bot Discord simple et efficace conçu pour faciliter l’organisation des sessions Guild vs Guild (GvG) dans ta guilde. Grâce à lui, les membres peuvent rapidement voter pour leurs horaires disponibles, et les leaders peuvent lancer des sondages interactifs en un clic.

⚙️ Commandes

  • /setguild : Définit le nom de la guilde
  • /settimezone : Choisit le fuseau horaire (NA/EU)
  • /startpoll : Lance un sondage adapté au jour
  • /setlanguage : Change la langue des annonces (FR/EN)
  • /help : Affiche l’aide du bot

🌐 Langue

Le bot est configuré par défaut en anglais. Tu peux utiliser /setlanguage pour changer la langue vers le français.

💡 Exemple d’utilisation

Un leader utilise /startpoll, les membres réagissent avec des émojis selon leur disponibilité. Simple, rapide et efficace.

🛡️ Permissions requises

Lors de l'invitation du bot, assure-toi de lui accorder les permissions suivantes : envoyer des messages, gérer les réactions et lire les messages dans les canaux souhaités.
